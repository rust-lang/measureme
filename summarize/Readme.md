# summarize

Summarize is a tool to produce a human readable summary of `mesaurme` profiling data.

## Example

$ git clone https://github.com/rust-lang/regex.git

$ cd regex

$ cargo rustc -- -Z self-profile

$ summarize pid-{pid}
